About The Project:

Making a led blink ounce and then twice

Components and Supples:

  • breadboard
  • jumperwires
  • raspberry
  • 220 oms resistor
  • led
  • t cobber
  • wedge cable

APPS AND ONLINE SERVICES:

  • thony

    • circut

    conneting a led to a raspberry:

    connect the led to the gpio 27 of the t cobber. connect the 220 resistor to the gpio gnd pin of the t cobber

    code for making led blink once: 

    
  importRPi.GPIO as GPIO
  import time
  GPIO.setmode(GPIO.BCM)
  GPIO.output(27,GPIO.HIGH)
  time.sleep(1)
  GPIO.output(27,GPIO.LOW)
  GPIO.cleanup()

    code for make led blink more than once: 

    
  importRPi.GPIO as GPIO
  import time
  GPIO.setmode(GPIO.BCM)
  GPIO.setup(27,GPIO.OUT)
  GPIO.output(27,GPIO.HIGH)
  GPIO.output(27,GPIO.LOW)
  GPIO.output(27,GPIO.HIGH)
  GPIO.output(27,GPIO.LOW)
  GPIO.cleanup()
