About The Project:

Making a four led on once or on for ten times

Components and Supples:

  • breadboard
  • jumperwires
  • raspberry
  • 220 oms resistor
  • led
  • t cobber
  • wedge cable

APPS AND ONLINE SERVICES:

    • circut

    conneting a led to a raspberry:

    schmatic

    code for making four led blink once: 

    
  importRPi.GPIO as GPIO
  import time
  GPIO.setmode(GPIO.BCM)
  pins = [6,13,19,26]
  GPIO.setup(pins, GPIO.OUT)
  GPIO.output(pins,GPIO.HIGH)
  time.sleep(1)
  GPIO.output(pins,GPIO.LOW)
  GPIO.cleanup()

    code for make led blink more than once: 

    
  importRPi.GPIO as GPIO
  import time
  GPIO.setmode(GPIO.BCM)
  GPIO.setup(27,GPIO.OUT)
  GPIO.output(27,GPIO.HIGH)
  GPIO.output(27,GPIO.LOW)
  GPIO.output(27,GPIO.HIGH)
  GPIO.output(27,GPIO.LOW)
  GPIO.cleanup()
