Making different ound

The passive buzzer can make sound just like active buzzer. It can output different sound. It has two pins one for signal pin and gnd pin.

Components

  • Arduino uno/Arduino mega/Arduino due
  • passive buzzer
  • Breadboard
  • Jumperwires
  • Usb cable

Description

There is a function in arduino specifical for the passive buzzer.The tone function is used for a passive buzzer. It is used for telling the buzzer duration and frequency of the tone. Two ways to write tone()

  • tone(pin,frequency)
  • tone(pin,frequency,duration)

Schematics

Code 

          
  const int buzzer = 8;
  void setup(){
      pinMode(buzzer, OUTPUT);
  }
  void loop(){
      tone(buzzer,1000);
      delay(1000);
      noTone(buzzer);
      delay(1000);
  }
