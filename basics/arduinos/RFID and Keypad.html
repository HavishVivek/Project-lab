Fading Light

In this project, contains a RGB color LED.The RGB led has 4 pin but only 3 are for color and for gnd. The RGB can give out three color. The color are red,blue,green.

Components

  • Arduino uno/Arduino mega/Arduino due
  • RGB led
  • 4 220 ohm resistor
  • Breadboard
  • Jumperwires
  • Usb cable

Description

There is a modulation for swiching bewteen on time and off time. The pwm stand for pulse with modulation. The time on is call pulse width. The person can change the duty cycle for voltage to variy. We use the analog write function to use pwm.

Schematics

Code 

          

#include 

  #define REDPIN 3
  #define GREENPIN 5
  #define BLUEPIN 6
  
  AlaLedRgb rgbLed;
  
  void setup()
  {
    rgbLed.initPWM(REDPIN, GREENPIN, BLUEPIN);                 // initialize output pins
    rgbLed.setBrightness(0x66FF44);                            // calibrate white
    rgbLed.setAnimation(ALA_GLOW, 5000, alaPalRgb);  // set the animation
  }
  
  void loop()
  {
    rgbLed.runAnimation();  // run the animation indefinitely
  }

Conclusion

The coolest led in the world is called RGB led. This led can give out three different color. You can programm it to give one color after one or fade one use pwm.
