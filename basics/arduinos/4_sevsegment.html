Getting Started with Keypad

Build a basic project with ultra soni sensor. This the first time using a ultra sonic sensor.

Components

  • Breadboard
  • Jumperwires
  • Arduino uno/Arduino mega/Arduino due
  • Usb cable
  • ultra sonic sensor

Schematics

Description

The ultra sonic sensor look like a face but it is used for measuring distance of an object. The sensor can olny measure up to 400cm and start at 2cm.

Code 

          
            #include 
              SevSeg sevseg;
              
              void setup(){
                byte numDigits = 1;
                byte digitPins[] = {};
                byte segmentPins[] = {6, 5, 2, 3, 4, 7, 8, 9};
                bool resistorsOnSegment = true;
                byte hardwareCinfig = COMMON_CATHODE;
              
                sevseg.begin(hardwareCinfig,numDigits,digitPins,segmentPins,resistorsOnSegment);
                sevseg.setBrightness(100); 
              }
              
              void loop(){
                sevseg.setNumber(10);
                sevseg.refreshDisplay();
              }
